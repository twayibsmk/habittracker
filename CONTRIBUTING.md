# Contributing to Habits Tracker

Thanks for your interest in contributing! Here's how you can help.

## 🛠️ Development Setup

1. Fork and clone the repo
2. Run `npm install`
3. Set up Supabase (see [README](README.md#3-set-up-supabase))
4. Run `npm run dev`

## 📝 Guidelines

- Keep the single-file component architecture
- Follow the existing code style (inline styles + injected CSS keyframes)
- Test your changes with both authenticated and unauthenticated states
- Maintain the space-station aesthetic for any UI additions

## 🐛 Bug Reports

Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- Browser and OS info

## 💡 Feature Requests

Open an issue describing the feature, why it's useful, and any design ideas.

## 🔀 Pull Requests

1. Create a feature branch from `main`
2. Make your changes
3. Test thoroughly
4. Submit a PR with a clear description

## 📜 Code of Conduct

Be respectful. Be constructive. Keep it professional.
